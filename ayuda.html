    Ayuda de Navegación

    Este sitio web ha sido diseñado priorizando la accesibilidad y la sencillez. A continuación se describen las características disponibles para facilitar su uso.

    Estructura del sitio

    El sitio se compone de cinco secciones principales accesibles desde el menú superior. Para conocer la estructura completa, puede consultar el Mapa del Sitio.

    Atajos de Teclado y Accesibilidad

    • Navegación por teclado: Todos los elementos son accesibles mediante el tabulador. El foco visual se resalta claramente con un borde de color.
    • Tamaño de texto: El sitio utiliza unidades relativas. Puede aumentar el tamaño del texto usando las funciones de zoom de su navegador ('Ctrl' + '+').

    Compatibilidad

    Este sitio es compatible con navegadores modernos (Chrome, Firefox, Edge, Safari) y se adapta automáticamente a dispositivos móviles y tabletas.
